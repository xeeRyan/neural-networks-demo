# 神经网络算法展示 (Neural Network Demo)

这是一个包含多种经典神经网络算法实现的学习仓库。本项目通过 Jupyter Notebook (`.ipynb`) 的形式，演示了不同神经网络模型的构建、训练与预测过程。

## 项目包含内容

本项目目前包含以下神经网络实现：

*   **CNN 手写数字识别**: 使用卷积神经网络 (Convolutional Neural Network) 对 MNIST 手写数字数据集进行分类。
*   **LSTM 航空乘客预测**: 使用长短期记忆网络 (Long Short-Term Memory) 进行时间序列预测。
*   **RBF 糖尿病预测**: 使用径向基函数网络 (Radial Basis Function Network) 进行回归/分类分析。
*   **SOM 鸢尾花识别**: 使用自组织映射 (Self-Organizing Map) 对鸢尾花数据集进行聚类分析。
